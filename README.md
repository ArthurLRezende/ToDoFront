# ToDo Portifólio

Para testar esse projeto:
- Descompacte o arquivo
- Abra o terminal e instale todas as depências:
```markdown 
    npm i 
```
- Altere no arquivo .env para o endereço da API do BackEnd:
```markdown
    VITE_API_URL: http://localhost:{porta_backend_local}
    ou
    VITE_AI_URL: https://{Endereço do servidor}
```
- Rode o projeto: 
```markdown
    npm run dev
```
